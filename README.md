# Angular 4 Node Seed (Upgraded from Angular 2.0)
Seed Project for Angular 4, Webpack 2 and Node JS

#### install dependencies:
```javascript
$ npm install
```
#### run the app server:
```javascript
$ npm start
```
#### build the UI code:
```javascript
$ npm run build
```
#### For Production:
```javascript
$ npm run build:prod 
```
